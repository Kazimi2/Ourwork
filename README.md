# 鸦片战争历史教育网页

一个内容严谨、风格简洁、支持多端适配的专题网页，用于思想政治作业展示。

## 项目特点

- **内容严谨**：史料准确，引用权威来源（如《中国近代史纲要》、权威学术文章）
- **设计简洁**：学术感设计，配色以深蓝、白色、灰色为主
- **响应式布局**：支持电脑、平板、手机多端适配
- **交互功能**：包含时间轴导航、互动答题等模块
- **无版权问题**：使用图标替代图片，避免版权问题

## 网页结构

### 9个核心模块

1. **导航栏** - 固定在顶部，支持移动端汉堡菜单
2. **首页横幅** - 大图背景，居中标题和简短引言
3. **核心概述** - 鸦片战争的定义、时间、原因、影响
4. **时间轴模块** - 详细时间线，包含10个关键事件（精确到日期）
5. **关键人物简介** - 林则徐、道光皇帝、查理·义律、关天培
6. **事件发展分析** - 战争阶段分析，相关史料说明
7. **互动答题模块** - 5道选择题，提交后显示得分与解析
8. **思考与启示** - 战争影响分析，讨论题
9. **页脚** - 声明、引用来源、小组信息

## 技术实现

### 文件结构
```
鸦片战争/
├── index.html              # 主页面
├── timeline.html           # 时间轴页面
├── styles/
│   ├── main.css           # 主样式文件
│   └── timeline.css       # 时间轴专用样式
├── scripts/
│   ├── main.js            # 主交互功能
│   ├── quiz.js            # 答题功能
│   └── timeline.js        # 时间轴交互
├── images/                # 图片目录（占位符）
├── test_quiz_functionality.html  # 答题功能测试
```

### 技术栈
- **HTML5** - 语义化标签结构
- **CSS3** - Flexbox/Grid布局，媒体查询响应式设计
- **JavaScript (ES6+)** - 交互功能实现
- **Font Awesome** - 图标库
- **Google Fonts** - 中文字体（思源宋体/黑体）

### 响应式设计
- 电脑端 (≥992px)：完整布局
- 平板端 (768px-991px)：适配布局
- 手机端 (≤767px)：移动优化布局

## 如何使用

### 本地运行
1. 下载所有文件到同一目录
2. 双击 `index.html` 在浏览器中打开主页面
3. 双击 `timeline.html` 打开时间轴页面

### GitHub Pages 部署
1. 将项目上传到GitHub仓库
2. 进入仓库设置 → Pages
3. 选择主分支作为源
4. 保存后访问提供的URL即可

### 测试功能
- `test_quiz_functionality.html` - 测试答题功能
- `final_test.html` - 完整功能测试报告
